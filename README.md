# Digital Design & Verification Portfolio
This repository serves as a centralized hub for my academic labs and ongoing engineering projects. It tracks my progress in RTL design and hardware verification, currently focusing on the MIPS architecture and AXI4 protocol.

Note: The projects in this repository are Work in Progress (WIP). I am actively developing the design logic and verification environments.

📂 Repository Structure
Projects/: Houses major design and verification projects.

MIPS Design: A MIPS-based processor design using Verilog/SystemVerilog, with Python integrated for scripting and automation tasks.

AXI4 Verification: A UVM-based verification environment for an AXI4 Slave interface.

Labs/: A collection of technical exercises and hardware modules developed during my studies and verification diploma.

🛠️ Ongoing Projects
1. MIPS Processor Design (In Progress)
A hardware implementation of the MIPS architecture.

Languages: Verilog, SystemVerilog, Python.

Status: Active Development. Currently focusing on the data path and instruction set implementation.

Integration: Utilizing Python for automated test vector generation.

2. AXI4 Slave Verification (In Progress)
Building a robust UVM (Universal Verification Methodology) environment to verify an AXI4 Slave.

Languages: SystemVerilog, UVM.

Status: Active Development.

Current Focus: Finalizing the base sequences for burst access and error injection.

💻 Technical Stack
HDLs: SystemVerilog, Verilog

Verification: UVM (Universal Verification Methodology)

Scripting: Python

Tools: Siemens QuestaSim, Vivado, VS Code

Environment: Linux (Ubuntu)
